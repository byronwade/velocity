# 17. Integrated Browser and Agent Platform

> Part of the [Velocity Platform Research & Integration Roadmap](README.md). Research snapshot: **15 July 2026**.

Current VS Code combines a real integrated browser with DevTools, debugging, scoped storage, remote proxying, AI context capture, session-isolated agent tools, planning, approvals, and multi-session agent surfaces. This is the correct native foundation for Velocity's Preview and browser-execution layers.
